- Resumen -

MIT App Inventor es una herramienta visual que permite crear aplicaciones móviles de manera intuitiva, lo que la hace perfecta para quienes están comenzando en el desarrollo. Una de sus características más destacadas es la posibilidad de conectarse mediante Bluetooth con dispositivos como Arduino, lo cual permite manejar elementos electrónicos como servomotores, LEDs y sensores. Utilizando bloques de programación, se puede establecer la comunicación entre la aplicación y módulos Bluetooth como el HC-05, lo que abre la puerta a proyectos interactivos en campos como la automatización y la robótica. - Introducción -

El Bluetooth es una tecnología inalámbrica de corto alcance que permite conectar dispositivos sin necesidad de una red externa. Mediante una app móvil, facilita el control remoto de sistemas en tiempo real, mejorando la interacción y experiencia del usuario.

- Materiales -

Cables de conexion

- Desarrollo -

Para crear la aplicación, se empleó MIT App Inventor, una plataforma visual que permite desarrollar apps de forma intuitiva mediante bloques, sin escribir código tradicional. Esta herramienta facilita procesos como la recolección de datos y su envío al Arduino vía Bluetooth, logrando una comunicación eficiente y un control efectivo del sistema.

Simulación

Diseño

La aplicación incluye cinco botones principales: conectar y desconectar el Bluetooth, encender y apagar el LED, y cerrar la aplicación. Tras establecer la conexión con el módulo HC-05, los botones de encendido y apagado envían los valores '1' y '0'. El Arduino interpreta estas señales y controla el estado del LED según el último comando recibido, manteniéndolo encendido o apagado.

- Resultados -

MIT App Inventor permite la comunicación por Bluetooth mediante bloques específicos que simplifican el intercambio de datos entre una app móvil y un dispositivo externo. Para ello, se añade el componente BluetoothClient desde la sección de conectividad, se empareja el dispositivo y se establece la conexión con el módulo Bluetooth. Una vez conectados, es posible enviar y recibir datos usando bloques de programación, lo que permite controlar elementos como servomotores, LEDs o sensores desde la aplicación.

- Conclusiones -

En conclusión, el uso de MIT App Inventor junto con la tecnología Bluetooth ofrece una solución accesible y eficaz para el desarrollo de aplicaciones móviles orientadas al control de dispositivos electrónicos. Gracias a su interfaz visual basada en bloques, es posible establecer una comunicación sencilla entre un smartphone y componentes como Arduino, facilitando la creación de proyectos interactivos en áreas como la robótica, la automatización y la educación. Esta combinación permite a los usuarios diseñar sistemas funcionales de manera intuitiva, fomentando el aprendizaje y la innovación tecnológica.
