- Resumen -

Este proyecto tiene como finalidad desarrollar una página web sencilla con HTML y publicarla en línea mediante GitHub Pages, aprendiendo así los pasos básicos para estructurar un sitio web, crear un repositorio y realizar su despliegue.

- Introducción -

En esta práctica se aprendió a desarrollar y publicar páginas web usando GitHub, comprendiendo que una página web es un documento digital estructurado con HTML y accesible desde un navegador. Este conocimiento proporciona una base útil para integrar interfaces web en proyectos futuros, permitiendo presentar y gestionar información de forma clara y eficiente.

- Desarrollo -

Para desarrollar y publicar una página web básica en HTML usando GitHub, primero crea un repositorio y agrega un archivo index.html con la estructura de tu sitio. Luego, activa GitHub Pages desde la configuración del repositorio, eligiendo la rama main como fuente. GitHub generará una URL pública desde la cual podrás visualizar tu página web en cualquier navegador.

Simulación

A diferencia de una simulación, en esta práctica la página web se desarrolló escribiendo el código manualmente desde un editor de texto. Luego, el archivo index.html fue abierto directamente en un navegador, lo que permitió visualizar de inmediato los resultados y cambios realizados, sin recurrir a emuladores o entornos de simulación.

Diseño

El diseño de la página web se mantuvo simple, enfocándose en una estructura clara y funcional. Para su desarrollo, se utilizaron diversos elementos de HTML que ayudaron a organizar el contenido de forma comprensible. Se emplearon párrafos (p) para presentar el texto, así como títulos (h1, h2, h3) para jerarquizar la información. También se integraron imágenes (img) para enriquecer visualmente la página y enlaces (a) que permitieron una navegación fluida entre diferentes secciones o páginas.

Construcción

El desarrollo de la página web se realizó escribiendo manualmente el código HTML en un editor de texto, sin utilizar herramientas de simulación o generación automática. Para estructurar el contenido de forma clara, se emplearon etiquetas como "h1" a "h4" para definir títulos y subtítulos, "p" para redactar los párrafos del texto, "img" para insertar imágenes, y "a" para crear enlaces que permiten navegar entre la página principal y una secundaria. Esta metodología permitió entender de manera práctica cómo se construyen y organizan los elementos básicos de una página web.

Publicar una página web básica en HTML usando GitHub Pages es un proceso accesible y sin costo.
