- Resumen -

Máquina CNC desarrollada con el propósito de automatizar la realización de dibujos con un iman doble que se mueve sobre una superficie con arena , operando a través de un sistema computarizado que permite un control preciso y repetible del proceso.

- Introducción -

El objetivo principal fue desarrollar un sistema capaz de ejecutar dibujos automatizados sobre una superficie plana, utilizando comandos enviados desde una computadora. La máquina fue diseñada para interpretar y ejecutar trayectorias con alta precisión gracias al uso de motores paso a paso, los cuales son controlados por una placa electrónica que actúa como intermediaria entre el software de control y los componentes mecánicos. Esta configuración permite que la CNC siga instrucciones específicas para crear figuras y patrones sobre un medio físico, como arena, papel o cualquier otra superficie adecuada.

- Desarrollo -

El proceso inició con el diseño mecánico de la máquina CNC, considerando el área de trabajo y la disposición de los ejes. Se cortaron piezas en MDF para la estructura y se imprimieron en 3D soportes y guías para los componentes. Luego, se ensamblaron los ejes X e Y, integrando motores paso a paso, poleas y correas para lograr un movimiento preciso.Tras el ensamblaje mecánico, se realizó el cableado y conexión de los motores a una placa Arduino con una shield para drivers. Finalmente, se estableció la comunicación serial con la computadora, desde donde se enviaron instrucciones de movimiento, permitiendo trazar dibujos de forma remota y automatizada.

Diseño

Se utilizo mdf para hecer toda la caja de adentro, para la caj de arena se uso acrilico de 5 milimetros para que el iman se puedea mover mas facil

Construcción

Se ensamblaron los motores paso a paso con sus respectivas poleas dentadas y se instaló la correa de distribución entre los ejes. El eje X es el encargado del movimiento, mientras que el eje Y permanece fijo. Para el control del sistema, se utilizó una CNC Shield montada sobre una placa Arduino, junto con dos drivers para los motores. Los cables de los motores se conectaron a los terminales correspondientes a los ejes X y Y en la shield, permitiendo un control eficiente del desplazamiento.

- Resultados -

Durante la prueba, fue capaz de desplazarse con precisión sobre la superficie de trabajo y realizar el trazado de una figura en la arena, demostrando así que tanto la programación como el ensamblaje mecánico están funcionando correctamente. Este logro confirma que los motores, el sistema de control y la interfaz de usuario están bien sincronizados y permite continuar con las siguientes fases del proyecto, como la mejora en la precisión del trazo y la realización de figuras más complejas.

- Conclusiones -

Se logró satisfactoriamente el objetivo de la práctica al ensamblar y controlar correctamente el sistema CNC. La integración de motores, poleas y la placa Arduino CNC Shield permitió realizar movimientos precisos, como trazar un cuadrado con exactitud. Esto demuestra la eficacia del control mediante Arduino para automatizar procesos y asegurar un funcionamiento confiable del sistema.
